## **Descripción**

Proyecto final para la materia Plataformas de Desarrollo.  
Se desarrolló un e-commerce completo con:

- Catálogo de productos
- Autenticación de usuarios
- Panel de administración
- Carrito de compras
- Lista de favoritos
- Gestión de pedidos
